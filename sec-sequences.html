    Section 2.1 Sequences

    Subsection 2.1.1 Definitions

    An infinite sequence is a function whose domain is, up to order isomorphism, the ordered set \((\N, \le)\text{.}\) Since we will only consider infinite sequences here, we simply call them sequences. If \(x\) is a sequence, we write \(x_n\) for its value at \(n\in\N\) and call \(x_n\) the \(n\)-th term of the sequence. We denote the sequence by \((x_n)\text{.}\)
    🔗
    Since a real sequence is a function from \(\N\) to \(\R\text{,}\) notions such as bounded above, bounded below, bounded, increasing, and decreasing apply exactly as they do for functions. An increasing (resp. a decreasing) sequence is bounded below (resp. above) by its first term. A sequence is constant if and only if it is both increasing and decreasing. We say that \((x_n)\) is strictly increasing if \(x_m \lt x_n\) whenever \(m \lt n\text{.}\) The concept strictly decreasing is defined similarly. We often regard real numbers as constant sequences.
    🔗

    Example 2.1.1.

    • \((1/n)\) is decreasing and bounded.
      🔗
    • \((n)\) is increasing and bounded below, but not bounded above.
      🔗
    • \(((-1)^n)\) is bounded, but neither increasing nor decreasing.
      🔗
    🔗
    A real sequence \((x_n)\) is called null if for every \(\varepsilon>0\) there exists \(N\in\N\) such that
    \begin{equation*} |x_n|<\varepsilon \quad\text{for all } n\ge N. \end{equation*}
    🔗
    Equivalently, \((x_n)\) is null if for every \(\varepsilon>0\text{,}\) the inequality \(|x_n|<\varepsilon\) holds for all but finitely many \(n\text{.}\)
    🔗

    Example 2.1.2.

    The sequence \((1/n)\) is null.
    🔗
    🔗
    We say that a sequence \((x_n)\) converges to \(L\in\R\text{,}\) written \(x_n\to L\) or \(\lim_n x_n = L\text{,}\) if the sequence \((x_n-L)\) is null. In that case \(L\) is called a limit of the sequence. A sequence is convergent if it has a limit. A sequence is divergent if it does not converge.
    🔗

    Example 2.1.3.

    Here are a few examples of tails:
    🔗
    • If \(x_n=n\text{,}\) then the 0-tail is \((n)\text{,}\) the 1-tail is \((n+1)\text{,}\) and the 3-tail is \((n+3)\text{.}\)
      🔗
    • If \(x_n=1/n\text{,}\) then the 2-tail is \((1/(n+2))\text{.}\)
      🔗
    • If \(x_n=(-1)^n\text{,}\) then the 1-tail is \(((-1)^{n+1})\text{,}\) while the 2-tail is \(((-1)^{n+2})=((-1)^n)\text{.}\)
      🔗
    🔗
    Different tails of the same sequence need not be equal, but the next proposition shows that they always have the same convergence behavior.
    🔗

    Proposition 2.1.4.

    A sequence and any of its tails have the same convergence behavior. More precisely, either both diverge, or both converge and have the same limit.
    🔗
    🔗

    Proof.

    Suppose \(x_n \to L\) and \(K \ge 0\text{.}\) Then for any \(\varepsilon \gt 0\text{,}\) there exists \(N \in \N\) such that \(|x_n - L| \lt \varepsilon\) for all \(n \ge N\text{.}\) But if \(n \ge N\text{,}\) then so is \(n+K\text{.}\) Therefore, \(|x_{n+K} - L| \lt \varepsilon\) for all \(n \ge N\text{.}\) This shows that \(x_{n+K} \to L\text{.}\)
    🔗
    Suppose \((x_n)\) is divergent but a tail of \((x_n)\) converges, say \(x_{n+K} \to L\) for some \(L\) and \(K \ge 0\text{.}\) Then for any \(\varepsilon \gt 0\text{,}\) there exists \(N \in \N\) such that \(|x_{m+K} - L| \lt \varepsilon\) whenever \(m \ge N\text{.}\) But that means whenever \(n \ge N+K\text{,}\) we have \(|x_n - L| \lt \varepsilon\text{.}\) This shows that \(x_n \to L\text{.}\)
    🔗
    🔗
    The following corollary is now clear.
    🔗

    Corollary 2.1.5.

    If a tail of one sequence is equal to a tail of another sequence, then the two sequences have the same convergence behavior.
    🔗
    🔗
    🔗

    Subsection 2.1.2 Fundamental Results

    Before going through some fundamental results about sequences, we introduce a handy shorthand: Instead of saying "there exists \(N\) such that for all \(n \ge N\)", we often suppress the reference to \(N\) (which may depend on other quantities) and say one of the following phrases:
    🔗
    • for all sufficiently large \(n\text{,}\)
      🔗
    • for all but finitely many \(n\text{,}\)
      🔗
    • for all \(n \gg 0\text{,}\)
      🔗
    • for all \(n \sim \infty\text{.}\)
      🔗
    With this convention, the statement that \((x_n)\) is null becomes: for every \(\varepsilon \gt 0\text{,}\) we have \(|x_n| \lt \varepsilon\) for all \(n \gg 0\text{.}\)
    🔗
    For \(K \ge 0\text{,}\) the \(K\)-tail of a sequence \((x_n)\) is the sequence \((x_{n+K})\text{.}\) So, the 0-tail of a sequence is the sequence itself. A tail of a sequence is simply a \(K\)-tail for some \(K \ge 0\text{.}\)
    🔗

    Proposition 2.1.6.

    Let \((x_n)\) and \((y_n)\) be real sequences.
    🔗
    1. \((x_n)\) is null if and only if \((-x_n)\) is null if and only if \((|x_n|)\) is null.
      🔗
      🔗
    2. If \((x_n)\) is null and \((y_n)\) is bounded, then \((x_ny_n)\) is null.
      🔗
      🔗
    3. If \((x_n)\) and \((y_n)\) are null, then \((x_n+y_n)\) is null.
      🔗
      🔗
    🔗

    Proof.

    For (1), note that \(|-x_n|=|x_n|\text{,}\) so the three conditions are equivalent.
    🔗
    For (2), choose \(B \gt 0\) with \(|y_n|\le B\) for all \(n\text{.}\) For any \(\varepsilon \gt 0\text{,}\) \(|x_n| \lt \varepsilon\) and hence \(|x_n y_n| \le B|x_n| \le B\varepsilon\) for all but finitely many \(n\text{.}\) Since \(\varepsilon \gt 0\) is arbitrary, so is \(B\varepsilon\text{.}\) This shows that the sequence \((x_ny_n)\) is null.
    🔗
    For (3), given \(\varepsilon>0\text{,}\) choose \(N_1,N_2\) so that \(|x_n|<\varepsilon/2\) for \(n\ge N_1\) and \(|y_n|<\varepsilon/2\) for \(n\ge N_2\text{.}\) Then for \(n\ge \max\{N_1,N_2\}\text{,}\) the triangle inequality gives \(|x_n+y_n|\le |x_n|+|y_n|<\varepsilon\text{.}\)
    🔗
    🔗

    Proposition 2.1.7.

    A convergent sequence has a unique limit.
    🔗
    🔗

    Proof.

    Suppose \(x_n\to L\) and \(x_n\to L'\text{.}\) Then \((L-x_n)\) and \((x_n-L')\) are null sequences, so by Proposition 2.1.6 their sum \((L-L')\) is also null.
    🔗
    A constant sequence is null only when its constant value is zero. Therefore \(L-L'=0\text{,}\) so \(L=L'\text{.}\)
    🔗
    🔗

    Proposition 2.1.8.

    Every convergent sequence is bounded.
    🔗
    🔗

    Proof.

    Suppose \(x_n\to L\text{.}\) Then there exists \(N\in\N\) such that whenever \(n \ge N\text{,}\) \(|x_n-L|<1\text{,}\) and so by the triangle inequality,
    \begin{equation*} |x_n| \lt |x_n-L| + |L| \lt 1+|L|. \end{equation*}
    🔗
    Hence \(|x_n|\le B:=\max\{|x_1|,\dots,|x_N|,|L|+1\}\) for every \(n\text{,}\) so \((x_n)\) is bounded.
    🔗
    🔗

    Theorem 2.1.9. Monotone Convergence Theorem.

    Every bounded increasing sequence converges to the supremum of its set of terms, and every bounded decreasing sequence converges to the infimum of its set of terms.
    🔗
    🔗

    Proof.

    We prove the increasing case. In this case, the set of terms \(S=\{x_n:n\in\N\}\) is bounded above and nonempty; let \(s\) be its supremum.
    🔗
    For any \(\varepsilon>0\text{,}\) \(s-\varepsilon\) is not an upper bound of \(S\text{,}\) so there exists \(N\) with \(s-\varepsilon<x_N\text{.}\) Because the sequence is increasing, whenever \(n \ge N\text{,}\)
    \begin{equation*} s-\varepsilon<x_N\le x_n\le s \lt s + \varepsilon. \end{equation*}
    Therefore \(|x_n-s|<\varepsilon\) for all \(n\ge N\text{,}\) so \(x_n\to s\text{.}\)
    🔗
    The decreasing case follows by applying the same argument to \((-x_n)\text{,}\) or by a direct dual proof using infima.
    🔗
    🔗
    The next proposition shows that the usual algebraic operations on convergent sequences again produce convergent sequences, and that taking limits commutes with these operations.
    🔗

    Proposition 2.1.10.

    Suppose \(a_n\to A\) and \(b_n\to B\text{.}\) Then:
    🔗
    1. \(a_n+b_n\to A+B\text{,}\)
      🔗
      🔗
    2. \(a_nb_n\to AB\text{,}\)
      🔗
      🔗
    3. \(1/a_n\to 1/A\text{,}\) provided \(A\ne0\) and \(a_n\ne0\) for all sufficiently large \(n\text{.}\)
      🔗
      🔗
    🔗

    Proof.

    Since \((a_n-A)\) and \((b_n-B)\) are null, Proposition 2.1.6 gives \((a_n+b_n-(A+B))=((a_n-A)+(b_n-B))\) null. Hence \(a_n+b_n\to A+B\text{.}\)
    🔗
    Also \(a_nb_n-AB = a_n(b_n-B) + (a_n-A)B.\) By Proposition 2.1.8, the sequence \((a_n)\) is bounded. Therefore \((a_n(b_n-B))\) is null, and since the constant sequence \((B)\) is bounded, \(((a_n-A)B)\) is also null. Their sum is null, so \(a_nb_n\to AB\text{.}\)
    🔗
    For the reciprocal, choose \(N_0\) such that \(|a_n-A|<|A|/2\) for all \(n\ge N_0\text{.}\) Then for \(n\ge N_0\text{,}\) \(|a_n|\ge |A|-|a_n-A|>|A|/2\text{,}\) so in particular \(a_n\ne0\text{.}\) Hence \(\left|\frac1{a_n}-\frac1A\right| = \frac{|a_n-A|}{|a_n||A|} \le \frac{2}{|A|^2}|a_n-A|.\) Since \((a_n-A)\) is null, it follows that \(1/a_n\to 1/A\text{.}\)
    🔗
    🔗

    Proposition 2.1.11. Squeeze Lemma.

    Suppose \(a_n\le x_n\le b_n\) for all \(n\in\N\) and \(a_n\to L\text{,}\) \(b_n\to L\text{.}\) Then \(x_n\to L\text{.}\)
    🔗
    🔗

    Proof.

    Let \(\varepsilon>0\text{.}\) Since \(a_n\to L\) and \(b_n\to L\text{,}\) there exists \(N\) such that for all \(n\ge N\) we have \(-\varepsilon<a_n-L<\varepsilon\) and \(-\varepsilon<b_n-L<\varepsilon\text{.}\)
    🔗
    Using \(a_n-L\le x_n-L\le b_n-L\text{,}\) we obtain \(-\varepsilon<x_n-L<\varepsilon\) for all \(n\ge N\text{.}\) Therefore \(x_n\to L\text{.}\)
    🔗
    🔗

    Proposition 2.1.12.

    Let \(S\subseteq\R\) be nonempty. If \(S\) is bounded above, then there exists an increasing sequence \((x_n)\) in \(S\) such that \(x_n\to \sup S\text{.}\) If \(S\) is bounded below, then there exists a decreasing sequence \((y_n)\) in \(S\) such that \(y_n\to \inf S\text{.}\)
    🔗
    🔗

    Proof.

    We prove the bounded-below case. Let \(a=\inf S\text{.}\) If \(a\in S\text{,}\) then the constant sequence \(y_n=a\) has the required properties.
    🔗
    Now assume \(a\notin S\text{.}\) Choose \(y_1\in S\text{.}\) Inductively, suppose \(y_k\in S\) has been chosen. Since \(a<(a+y_k)/2<y_k\text{,}\) the number \((a+y_k)/2\) is not a lower bound of \(S\text{.}\) Thus there exists \(y_{k+1}\in S\) with \(a<y_{k+1}<(a+y_k)/2<y_k\text{.}\)
    🔗
    This constructs a decreasing sequence in \(S\) with \(0<y_{n+1}-a<(y_n-a)/2\) for all \(n\text{.}\) By induction, \(0<y_n-a\le (y_1-a)/2^{n-1}\text{.}\) Since the right-hand side tends to \(0\text{,}\) the squeeze lemma gives \(y_n\to a=\inf S\text{.}\)
    🔗
    The bounded-above case is proved similarly, or by applying the bounded-below case to \(-S\text{.}\)
    🔗
    🔗

    Proposition 2.1.13.

    If \(x_n\to L\) and \(x_n\ge0\) for all \(n\text{,}\) then \(L\ge0\text{.}\)
    🔗
    🔗

    Proof.

    Suppose instead that \(L<0\text{.}\) Then for every \(n\text{,}\) \(x_n-L\ge -L=|L|\text{,}\) so \(|x_n-L|\ge |L|>0\text{.}\) This contradicts \(x_n\to L\text{.}\) Therefore \(L\ge0\text{.}\)
    🔗
    🔗

    Corollary 2.1.14.

    If \(x_n\to x\text{,}\) \(y_n\to y\text{,}\) and \(x_n\le y_n\) for all \(n\text{,}\) then \(x\le y\text{.}\)
    🔗
    🔗

    Proof.

    The sequence \((y_n-x_n)\) converges to \(y-x\) by Proposition 2.1.10, and each term is nonnegative. Hence \(y-x\ge0\) by Proposition 2.1.13, so \(x\le y\text{.}\)
    🔗
    🔗
    🔗

    Subsection 2.1.3 Subsequences

    A subsequence of a sequence \((x_n)\) is a sequence of the form \((x_{n_k})\text{,}\) where \((n_k)\) is a strictly increasing sequence of natural numbers.
    🔗

    Proposition 2.1.15.

    Every subsequence of a convergent sequence converges to the same limit.
    🔗
    🔗

    Proof.

    Suppose \(x_n\to L\) and \((x_{n_k})\) is a subsequence. Since \((n_k)\) is strictly increasing, we have \(n_k\ge k\) for every \(k\text{.}\)
    🔗
    Let \(\varepsilon>0\text{.}\) Choose \(N\) such that \(|x_n-L|<\varepsilon\) for all \(n\ge N\text{.}\) If \(k\ge N\text{,}\) then \(n_k\ge k\ge N\text{,}\) so \(|x_{n_k}-L|<\varepsilon\text{.}\) Hence \(x_{n_k}\to L\text{.}\)
    🔗
