    Section 1.2 The Completeness Axiom

    The following axiom is what sets the real field apart from other ordered fields.
    🔗

    Axiom 1.2.1. The Completeness Axiom of \(\R\).

    Every non-empty set of real numbers that is bounded above has a supremum in \(\R\text{.}\)
    🔗
    🔗
    For an ordered set, the least upper bound property means: every non-empty subset that is bounded above has a least upper bound (that is, a supremum) in the set. So the completeness axiom says that \(\R\text{,}\) with its usual order, has the least upper bound property.
    🔗

    Subsection 1.2.1 Consequences

    Proposition 1.2.2.

    The completeness axiom of \(\R\) is equivalent to: every non-empty subset of \(\R\) that is bounded below has an infimum in \(\R\text{.}\)
    🔗
    🔗

    Proof.

    Let \(A\subseteq\R\) be non-empty and bounded below. Then \(-A\) is non-empty and bounded above, so \(s:=\sup(-A)\) exists. We claim \(-s=\inf A\text{.}\) For \(a\in A\text{,}\) we have \(-a\in -A\text{,}\) hence \(-a\le s\text{,}\) so \(-s\le a\text{.}\) Thus \(-s\) is a lower bound of \(A\text{.}\) If \(l\) is any lower bound of \(A\text{,}\) then \(-l\) is an upper bound of \(-A\text{,}\) so \(s\le -l\text{,}\) i.e. \(l\le -s\text{.}\) Therefore \(-s\) is the greatest lower bound of \(A\text{.}\)
    🔗
    Conversely, assume every non-empty bounded-below subset of \(\R\) has an infimum. Let \(B\subseteq\R\) be non-empty and bounded above. Then \(-B\) is non-empty and bounded below, so \(m:=\inf(-B)\) exists. The same argument shows \(-m=\sup B\text{.}\) Hence the least upper bound property holds.
    🔗
    🔗

    Checkpoint 1.2.3.

    Suppose \(\emptyset\ne A\subseteq B\subseteq\R\) and that \(B\) is bounded above. Show that
    \begin{equation*} \inf B \le \inf A \le \sup A \le \sup B. \end{equation*}
    🔗
    🔗

    Proposition 1.2.4.

    The set \(\N\) is not bounded above in \(\R\text{.}\)
    🔗
    🔗

    Proof.

    Suppose \(\N\) were bounded above. By the least upper bound property, \(u:=\sup\N\) exists. Since \(u-1<u\text{,}\) the number \(u-1\) cannot be an upper bound of \(\N\text{.}\) So there is \(n\in\N\) with \(u-1<n\text{.}\) Then \(u<n+1\text{,}\) and \(n+1\in\N\text{,}\) contradicting that \(u\) is an upper bound.
    🔗
    🔗

    Theorem 1.2.5. Existence of Positive Square Roots.

    For each \(a>0\text{,}\) there exists a unique \(u>0\) such that \(u^2=a\text{.}\)
    🔗
    🔗

    Proof.

    Uniqueness is immediate because \(x\mapsto x^2\) is strictly increasing on \((0,\infty)\text{.}\)
    🔗
    For existence, define \(S=\{x\in\R : x>0,\ x^2\le a\}\text{.}\) The set is non-empty: if \(a\ge 1\text{,}\) then \(1\in S\text{;}\) if \(0<a<1\text{,}\) then \(a\in S\text{.}\) It is bounded above by \(M=\max\{a,1\}\text{,}\) since \(x>M\) implies \(x^2>a\text{.}\) By completeness, \(u:=\sup S\) exists.
    🔗
    If \(u^2<a\text{,}\) then for sufficiently small \(\varepsilon>0\text{,}\) we still have \((u+\varepsilon)^2\le a\text{,}\) so \(u+\varepsilon\in S\text{,}\) contradicting that \(u\) is an upper bound of \(S\text{.}\) If \(u^2>a\text{,}\) then for sufficiently small \(\varepsilon>0\text{,}\) \((u-\varepsilon)^2>a\text{;}\) then every \(x\in S\) satisfies \(x\le u-\varepsilon\text{,}\) so \(u-\varepsilon\) is an upper bound of \(S\text{,}\) contradicting leastness of \(u\text{.}\) Hence \(u^2=a\text{.}\)
    🔗
    🔗
    A standard comparison example is the ordered field \(\Q\text{:}\) the set \(\{q\in\Q:q>0,\ q^2<2\}\) is non-empty and bounded above in \(\Q\text{,}\) but it has no supremum in \(\Q\text{.}\) Therefore \(\Q\) does not satisfy the least upper bound property.
