For sets \(A,B\text{,}\) \(|A|=|B|\) means there is a bijection between them. In that case, we say that \(A\) and \(B\) have the same cardinality. We write \(|A| \le |B|\) if there is an injection \(A \to B\text{.}\) A set \(A\) is countably infinite if \(|A|=|\N|\text{.}\) A set is countable if it is either finite or countably infinite.
