Definition E.1.1.

A set \(A \subseteq \R\) has measure zero if for every \(\varepsilon \gt 0\) there exists a sequence of open intervals \((I_k)_{k=1}^{\infty}\) such that
\begin{equation*} A \subseteq \bigcup_{k=1}^{\infty} I_k \qquad\text{and}\qquad \sum_{k=1}^{\infty} |I_k| \le \varepsilon, \end{equation*}
where \(|I_k|\) denotes the length of \(I_k\text{.}\) A set of measure zero is also called a null set.
