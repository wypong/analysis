An infinite sequence is a function whose domain is, up to order isomorphism, the ordered set \((\N, \le)\text{.}\) Since we will only consider infinite sequences here, we simply call them sequences. If \(x\) is a sequence, we write \(x_n\) for its value at \(n\in\N\) and call \(x_n\) the \(n\)-th term of the sequence. We denote the sequence by \((x_n)\text{.}\)
