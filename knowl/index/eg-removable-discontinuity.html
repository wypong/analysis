Example 4.2.7.

Define
\begin{equation*} f(x)=\begin{cases} \dfrac{x^2-1}{x-1} & \text{if } x \neq 1,\\ 0 & \text{if } x=1. \end{cases} \end{equation*}
For \(x \neq 1\text{,}\) we have \(f(x)=x+1\text{,}\) so \(\lim_{x\to 1} f(x)=2\text{.}\) However, \(f(1)=0\text{,}\) so \(f\) is not continuous at \(1\text{.}\) If we redefine the value at \(1\) to be \(2\text{,}\) then the resulting function becomes continuous at \(1\text{.}\) This is the basic model of a removable discontinuity.
