Proposition 4.4.1.

Every cluster point of a closed interval \([a,b]\) belongs to \([a,b]\text{.}\)
🔗
🔗

Proof.

For any \(c \lt a\text{,}\) let \(r=(a-c)/2\text{.}\) Then \((c-r,c+r) \subseteq (-\infty,a)\text{,}\) which is disjoint from \([a,b]\text{.}\) So \(c\) is not a cluster point of \([a,b]\text{.}\) The case \(c \gt b\) is similar, using \(r=(c-b)/2\text{.}\) Therefore every cluster point of \([a,b]\) must lie in \([a,b]\text{.}\)
