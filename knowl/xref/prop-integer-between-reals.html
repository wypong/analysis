Proposition 1.2.6.

For any \(x,y \in \R\text{,}\) if \(y-x > 1\text{,}\) then there exists \(m \in \Z\) such that \(x \le m < y\text{.}\)
🔗
🔗

Proof.

By Proposition 1.2.5, there exists \(n \in \Z\) such that \(n \le x < n+1\text{.}\)
🔗
If \(y \le n+1\text{,}\) then
\begin{equation*} n \le x < y \le n+1, \end{equation*}
so \(y-x < (n+1)-n = 1\text{,}\) contradicting the hypothesis. Therefore \(n+1 < y\text{.}\)
🔗
Let \(m=n+1\text{.}\) Then \(m \in \Z\) and \(x < m < y\text{.}\) In particular, \(x \le m < y\text{.}\)
